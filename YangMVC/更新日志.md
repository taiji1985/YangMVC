# 2020-7-5

## 1 添加了通过return输出的功能。如

```java
//相当于output("hello")        
public String retstr(){
	return "hello";
}
//相当于outputJSON(obj);
public Object retjson(){
	return MapHelper.toMap("yang",123,"wang",444);
}
//相当于renderFreemarker("/index.html")
public Object retfm(){
	return freemarker("/index.html");
}

```
